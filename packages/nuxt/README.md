## Documentation

You can find the documentation [on the website](https://vaxee.letstri.dev).

## Installation

```shell
npm i @vaxee/nuxt
```

## Usage

Add to `modules` in `nuxt.config.js`:

```ts
export default defineNuxtConfig({
  modules: ["@vaxee/nuxt"],
});
```
