# `@vaxee/nuxt`

> Nuxt 2 & 3 module

## Installation

```shell
npm i @vaxee/nuxt
```

## Usage

Add to `modules` (Nuxt 3) or `buildModules` (Nuxt 2) in `nuxt.config.js`:

```js
export default defineNuxtConfig({
  modules: ["@vaxee/nuxt"],
});
```
